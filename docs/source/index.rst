Welcome to iplotProcessing's documentation!
===========================================

**iplotProcessing** combines an expression parser and evaluator with a context manager to facilitate
mathemtical/user-defined time-base-aware operations upon signal-like data objects.

Check out the :doc:`getting-started` section for further information, including how to
:ref:`install <installation>` **iplotProcessing**.
